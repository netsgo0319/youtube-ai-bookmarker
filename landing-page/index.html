Problem

영상 인사이트가 사라지는 다섯 가지 이유

폴더·노션·북마크 등 여러 툴로 흩어진 영상 기록 “예전에 봤던 그 장면”을 텍스트처럼 검색할 방법 부재 수동 타임스탬프/클립 작업으로 반복되는 허드렛일 자동 태그·정리 도구의 낮은 정확도 팀 협업 컨텍스트가 영상에 반영되지 않음

결국 “영상 인사이트는 저장해도 다시 찾지 못한다”가 핵심 문제입니다.

Solution

영상도 텍스트처럼 검색하고 바로 쓰도록

제품 비전

지식노동자, 강사, 연구형 크리에이터가 영상에서 얻은 인사이트를 바로 찾고, 바로 쓰고, 바로 재사용할 수 있게 돕는 AI 기반 개인/팀용 영상 지식베이스입니다.

핵심 파이프라인

  1. 링크 입력
  2. 자동 텍스트화 및 요약
  3. 문장/키워드 검색
  4. 타임스탬프 점프 & 폴더 정리

차별화 포인트

  • 영상 전체를 문장 단위로 분해해 검색 정확도 확보
  • 검색 → 즉시 타임스탬프 이동으로 재생산 시간 단축
  • Inbox & 폴더 기본 구조로 빠른 정리

MVP Scope

Flow

동작 방식 4단계

  1. 01. 링크 입력

    YouTube URL을 넣으면 메타데이터와 자막을 확보합니다.

  2. 02. 텍스트화

    자막 또는 STT로 전체 텍스트를 문장 단위로 저장합니다.

  3. 03. 검색 & 요약

    Full-text 검색, 3·7점 요약, 자동 클립을 동시에 제공합니다.

  4. 04. 재사용

    검색 결과를 클릭해 타임스탬프로 이동하거나 폴더별로 정리합니다.

Tech stack

Next.js + Supabase + OpenAI

Next.js

서버 컴포넌트 기반으로 빠른 렌더링과 SEO를 확보합니다.

Supabase

영상 메타데이터, 텍스트, 타임스탬프를 안전하게 저장합니다.

OpenAI

Whisper로 STT, GPT로 요약과 클립 후보를 생성합니다.

YouTube iframe API

초 단위 타임스탬프 이동과 플레이어 제어를 제공합니다.
